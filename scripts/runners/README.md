# Runner Scripts

All `run_*.py` launcher scripts live here. Run from the **repo root**.

| Script | Purpose |
|--------|----------|
| `run_demo.py` | Quick demo / smoke-test run |
| `run_evolution.py` | Evolutionary strategy mutation loop |
| `run_godmode.py` | God-mode: all strategies + ensemble + ML |
| `run_maximum_evolution.py` | Maximum evolution pass |
| `run_optimize.py` | Optuna / hyperparameter optimisation |
| `run_paper.py` | **Primary paper-trading runner** |
| `run_paper_argus.py` | Paper run scoped to Argus strategies |
| `run_paper_test.py` | Paper run integration smoke test |
| `run_peak.py` | Peak performance mode |
| `run_pinnacle.py` | Pinnacle engine runner |
| `run_quantum_evolution.py` | Quantum-evolution combined run |
| `run_tests.bat` | Windows test runner wrapper |
| `run_ultimate.py` | **Primary production runner** |
| `run_ultimate_evolution.py` | Ultimate + evolution R&D |
| `run_validation_backtest.py` | CI validation backtest |

See [`ENTRYPOINTS.md`](../../ENTRYPOINTS.md) for canonical usage.
