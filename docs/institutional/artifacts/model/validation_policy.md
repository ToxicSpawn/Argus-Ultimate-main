# Model Validation Policy

Owner: Risk/Validation  
Review cadence: Monthly

## Requirements
1. Every strategy has a registry record with owner, limits, and failure modes.
2. Walk-forward and stress tests are required before promotion.
3. Validation reviewer must be independent of strategy author.
4. Promotion artifacts include report paths, hashes, and approval record.

## Minimum Promotion Evidence
1. Fresh walk-forward report meeting thresholds.
2. Soak gate pass report within age limit.
3. Change record with rollback procedure.
