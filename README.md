## Writing a Strategy

```python
from core.strategy.base_strategy import BaseStrategy
from core.strategy.strategy_registry import StrategyRegistry

class MyStrategy(BaseStrategy):
    name = "MyStrategy"

    async def on_bar(self, bar) -> None:
        if bar.close > bar.open * 1.001:
            await self.emit_signal(symbol=bar.symbol, side="buy", confidence=0.8)

    async def on_signal(self, signal) -> None:
        pass

# Register
StrategyRegistry().register(MyStrategy)
```

Load at runtime:
```bash
argus start --config argus.yaml
# Strategy auto-discovered from registry
```
