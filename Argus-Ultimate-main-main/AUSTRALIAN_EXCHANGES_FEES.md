## 🔧 SETTING UP FOR ZERO/LOW FEES

### **Step 1: Choose Exchange**
```
For Argus: Kraken (best API)
For manual: Coinspot or Independent Reserve (lowest fees)
For both: Use Independent Reserve (Sydney-based, low fees)
```

### **Step 2: Deposit Strategy**
```
Always use:
- PayID/Osko: FREE and instant
- Bank transfer: FREE (1-2 days)

Never use:
- Card deposits: 2-4% fees
```

### **Step 3: Trading Strategy**
```
To minimize fees:
1. Use limit orders (maker) when possible - lower fees
2. Batch trades (fewer large trades vs many small)
3. Hold positions longer (less trading = less fees)
4. Use exchange with volume discounts
```

### **Step 4: Withdrawal Strategy**
```
Most exchanges:
- AUD withdrawals: FREE
- Crypto withdrawals: Variable network fees

Best practice:
- Keep most funds in exchange for trading
- Only withdraw profits periodically
- Use AUD withdrawals (free) not crypto
```
