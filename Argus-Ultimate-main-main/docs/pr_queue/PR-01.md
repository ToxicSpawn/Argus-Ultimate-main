# PR-01 Audit Spine

## Scope
- Propagate `run_id` + `trace_id` through each trading cycle and candidate path.
- Add canonical `ReasonCode` enum for allow/reject/halt outcomes.
- Write a `decision_snapshots` SQLite row for every candidate decision (allowed or rejected).
- Persist gate outcomes and cost context for post-trade explainability.

## Primary Files
- `unified_trading_system.py`
- `execution/reason_codes.py`
- `execution/state_store.py`
- `monitoring/trade_ledger.py`

- `py -3 -m unittest tests_unified.test_pr_01_to_06_execution_controls.TestPR01ToPR06ExecutionControls.test_reconciliation_halt_reason_is_snapshotted -v`

## Query Example
