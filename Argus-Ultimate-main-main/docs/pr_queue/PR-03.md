# PR-03 Reconciliation Loop

## Scope
- Add periodic reconciliation between internal state and exchange/simulator truth.
- Apply configurable drift thresholds.
- Auto-correct small drift and halt on large drift.
- On large drift, emit `RECONCILIATION_HALT`, stop new intents, and cancel open orders.

- `unified_trading_system.py`
- `execution/state_store.py`
- `execution/reason_codes.py`
- `monitoring/alerting.py`
