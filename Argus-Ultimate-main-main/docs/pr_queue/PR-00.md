# PR-00 Guardrails + Windows Run Kit

## Scope
- KILL_SWITCH hard stop on trading commands.
- Live mode requires exact `YES` confirmation unless `--yes-live`.
- Auto-create `logs/` and `data/`.
- Strict config validation for unknown top-level keys and range checks.
- Windows scripts: `Install.ps1`, `Run-Smoke.ps1`, `Run-Soak.ps1`, `make.ps1`.
- Deterministic offline smoke mode via `--offline-smoke`.

## Primary Files
- `main.py`
- `core/config_manager.py`
- `make.ps1`
- `ops/windows/Install.ps1`
- `ops/windows/Run-Smoke.ps1`
- `ops/windows/Run-Soak.ps1`
- `README.md`
